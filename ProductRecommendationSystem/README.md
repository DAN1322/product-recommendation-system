# Product Recommendation System

## Overview
This project is a Product Recommendation System developed using Machine Learning techniques. It recommends products to users based on product similarity and hybrid filtering methods.

## Features
- Product search functionality
- Similar product recommendations
- Content-based filtering using TF-IDF Vectorization
- Cosine Similarity based recommendation engine
- Hybrid filtering approach for improved recommendations
- User-friendly web interface

## Technologies Used
- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML/CSS/Bootstrap

## Machine Learning Techniques
### 1. TF-IDF Vectorization
Converts product information into numerical vectors by measuring the importance of words.

### 2. Cosine Similarity
Calculates similarity between products based on vector representations.

### 3. Hybrid Filtering
Combines multiple recommendation approaches to provide more relevant product suggestions.
